### Green (Confirmed/Manual)
```css
background-color: rgb(240, 253, 244)  /* bg-green-50 */
border-color: rgb(74, 222, 128)       /* border-green-400 */
```

## Accessibility

### Color-Independent Identification
Each status has multiple indicators beyond color:
1. **Text labels**: "Manually Added", "Confirmed by User", "Rejected", "Unconfirmed"
2. **Badge indicators**: "AI" vs "Manual"
3. **Metadata fields**: Clear "Modified by" and "Confirmed by" text

## Mobile Responsive Behavior

### Desktop (> 768px)
- Full width cards
- Side-by-side metrics layout
- All information visible

### Tablet (768px - 1024px)
- Cards stack with slight padding reduction
- Metrics may wrap to 2 rows

### Mobile (< 768px)
- Full-width single column
- Metrics stack vertically
- Scrolling maintained at fixed height

## Performance Considerations

- **Rendering**: Only cached annotations displayed (not AI analysis anomalies)
- **Filtering**: Real-time filtering by source using Array.filter()
- **Updates**: Event-driven updates via custom events
- **Storage**: localStorage for persistence
- **Optimization**: Could memoize filtered lists if > 100 annotations
